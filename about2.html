*Hospedagem de Site:*

Voltar ao Índice

Hospedagem de site é o serviço que armazena e disponibiliza um site na Internet, tornando-o acessível aos visitantes.

Conceitos Relacionados à Hospedagem e Conectividade:

  • Multiplataforma: Compatível com diferentes sistemas operacionais, como Windows e Linux.
  • Certificado SSL:Garante uma conexão segura e criptografada de ponta a ponta (HTTPS). Indica que o site é confiável.
  • Domínio:Endereço registrado de um site, associado a um DNS (Sistema de Nomes de Domínio). Domínio “.br”: Todo site que termina com “.br” indica que é registrado no Brasil.
  • FTP (File Transfer Protocol)Protocolo de conexão usado para enviar e receber arquivos do servidor.
    1. Ao hospedar um site, são fornecidos: usuário, senha e endereço de conexão FTP.
    2. Para acesso web, usamos "www"; para transferência de arquivos, utilizamos o protocolo FTP.
    3. Arquivos como HTML são enviados ao servidor por meio de programas como o FileZilla.

Server-side × Client-side:

Voltar ao Índice

Server-side: São as linguagens usadas no servidor (JS, Php,C#,Java), cada linguagem tem a sua característica, aprenda pelo menos 1 linguagem e prefira a mais utilizada.

Client-side Linguagem usada do lado do cliente (html, CSS, JS)

O JS é usado tanto sever-side e Client-Side,a diferença é que no lado do servidor é pelo Node (programa) e no lado do cliente pelo navegador

HTML

Voltar ao Índice

HyperText Markup Language, ou em português, Linguagem de Marcação de Hipertexto.

  • Não é uma linguagem programação mas um arquivo de marcação.
  • Organiza o conteúdo da página por meio de tags, que delimitam textos, imagens, links, entre outros elementos.
  • Cada tag tem uma função específica que o navegador interpreta para exibir corretamente o conteúdo da web.
  • É a base da estrutura de qualquer site, servindo como esqueleto da página.
